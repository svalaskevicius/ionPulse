Detailed Description

The interface of the IonLayout plugin.

The LayoutPlugin interface provides an unified application scope access to the LayoutManager.

To use it, client plugins should list IonLayout::LayoutPlugin::name() in their dependencies. This way the client plugins will receive an instance of LayoutPlugin in their IonCore::BasicPlugin::addParent() method.
