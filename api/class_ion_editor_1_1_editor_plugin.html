Detailed Description

The interface of the IonEditor plugin.

The EditorPlugin interface provides an unified application scope access to the EditorWidgetBuilder.

To use it, client plugins should list EditorPlugin::EditorPlugin::name() in their dependencies. This way the client plugins will receive an instance of EditorPlugin in their IonCore::BasicPlugin::addParent() method.
