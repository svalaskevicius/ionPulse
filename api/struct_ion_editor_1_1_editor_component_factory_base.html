virtual QString IonEditor::EditorComponentFactoryBase::getIdentifier ( ) [pure virtual]

Identify this component factory.

This identifier is used to limit created components for one Editor to be one for each identifier. If the same identifier is available for several file types in the same hierarchy, only the most specific EditorComponent factory will be used.
