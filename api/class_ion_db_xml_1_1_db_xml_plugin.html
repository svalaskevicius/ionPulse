Detailed Description

The interface of the IonDbXml plugin.

The DbXmlPlugin interface provides an unified application scope access to Berkeley DB XML based information storage and management routines.

To use it, client plugins should list IonDbXml::DbXmlPlugin::name() in their dependencies. This way the client plugins will receive an instance of DbXmlPlugin in their IonCore::BasicPlugin::addParent() method.
