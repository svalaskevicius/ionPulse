Detailed Description

The interface of the IonProject plugin.

The ProjectPlugin interface provides an unified application scope access to modify TreeModelSource factory and set their own factory using a Decorator Pattern to extend the project tree functionality.

To use it, client plugins should list IonProject::ProjectPlugin::name() in their dependencies. This way the client plugins will receive an instance of ProjectPlugin in their IonCore::BasicPlugin::addParent() method.
