Detailed Description

Holds the bare window creation and plugin management functionality.

The IonCore module is the whole application. The main responsibilities of the module are:

  • create the QApplication and bare main window;
  • load and initialise plugins.

After the above steps all the functionality is deferred to the plugins, keeping the application layer thin.
